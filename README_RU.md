# Vietnam News Telegram Bot

Бот для получения вьетнамских новостей, их перевода на русский и публикации в Telegram-канал/чат по расписанию (09:00, 12:00, 15:00, 18:00 GMT+7). Администраторы могут добавлять рекламные блоки между новостями, а пользователи могут оставлять комментарии с фильтрацией нецензурной лексики (система предупреждений и бана).

## Особенности

- Получение вьетнамских новостей из новостного API (например, GNews)
- Перевод заголовков и содержания на русский язык через Google Translate (бесплатно)
- Публикация до 4 новостей за один запуск
- Рекламные блоки (только для администраторов) между новостями
- Комментарии пользователей с проверкой на мат (первое предупреждение, второй бан)
- База данных SQLite для хранения новостей, рекламы, пользователей и комментариев
- Docker-образ для лёгкого развертывания

## Установка

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/werewolf85/vn-news-bot2.git
   cd vn-news-bot2
   ```

2. Создайте файл `.env` со следующими переменными:
   ```
   BOT_TOKEN=ваш_токен_telegram_бота
   ADMIN_IDS=123456789,987654321   # список Telegram user ID через запятую
   NEWS_API_KEY=ваш_ключ_к_новостному_API   # например, GNews
   TIMEZONE_OFFSET=7               # GMT+7 для Вьетнама
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустите бота:
   ```bash
   python bot.py
   ```

   Или через Docker:
   ```bash
   docker compose up --build -d
   ```

## Использование

- После запуска бота отправьте любое сообщение (например, `/start`), чтобы бот сохранил ваш chat_id.
- Бот будет автоматически публиковать новости по расписанию.
- Администраторы могут добавить рекламный блок командой:
  ```
  /addad <текст> [ссылка_на_картинку]
  ```
- Пользователи могут оставить комментарий к последней новости, просто отправив сообщение; бот проверит его на нецензурную лексику.
- Используйте `/mywarn`, чтобы проверить количество ваших предупреждений.
